# How to include Bootstrap
1. Install Bootstrap for Vue
```
npm install bootstrap bootstrap-vue
```
2. Include in main script
```
import "bootstrap/dist/css/bootstrap.css" 
import "bootstrap-vue/dist/bootstrap-vue.css" 
```
